# Металлическая рама для промышленного оборудования

## Описание
Проект рамы для установки промышленного агрегата.
Учитывались вибрационные нагрузки и требования по жёсткости.

## Что сделано
- 3D-модель конструкции
- Комплект чертежей
- Расчёт прочности

## Используемые инструменты
- SolidWorks
- AutoCAD
- Excel
